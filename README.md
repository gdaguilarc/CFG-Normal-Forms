# CFG-Normal-Forms

> An algorithm that transforms any context free grammar to its Chomsky's normal form

## Installation & Run

```sh
npm install

npm start
```

## Usage example

Use one of the test files in ./test_files or a file ".txt" with the sintax that is specified below.
